# MIDI-Video-Freeloading
> $\color{#2ea44f}{\textsf{\textbf{이제 미디 파일과 비디오만 준비하세요!}}}$
> > https://midi-video-customize.vercel.app/

기존 DAW 기반 작업보다 더 간단하게 MIDI와 영상을 활용할 수 있습니다.
> [!IMPORTANT]
> 이 사이트를 사용해 생성된 영상 및 결과물은 자유롭게 녹화, 편집, 업로드 및 공유할 수 있습니다.
> 본 사이트 출처 표기는 필요하지 않습니다.
> 
> 단, 사용자가 업로드한 MIDI, 오디오, 영상 파일의 저작권은 해당 파일의 원 저작자에게 있으며,
> 해당 콘텐츠 사용에 대한 책임은 사용자에게 있습니다.

## 👉 주요 기능
- MIDI 파싱 + 피아노롤 표시
- 샘플러로 소리 생성
- 영상 제어 (좌우 반전 & 상하 반전 *실험적*)
- 오디오(WAV) 내보내기 기능
- 영상 효과 (페이드 아웃, 팝 등)

----
- *볼륨 조절 (추가 예정)*
- *다양한 영상 효과 추가 : 랜덤 회전 (아주 약하게), 확대 → 축소 → 확대 패턴(추가 예정)*
- *대사를 순차적으로 재생하는 기능(대사 하나 하나 잘라서 첨부하면, 다음 노트에는 첨부한 파일들이 순차적으로 재생)(구상 중)*

## 📖 사용 방법
> 1️⃣ 파일 준비

MIDI 파일 (.mid)

비디오(.mp4) 또는 오디오(.mp3)
> [!NOTE]
> MIDI는 재생 정보를 담당하고, 실제 소리는 오디오 파일을 기반으로 생성됩니다.
>
> 사용자가 업로드한 MIDI, 오디오, 비디오 파일은 서버로 전송되지 않으며 브라우저에서만 처리됩니다.

----

> 2️⃣ 파일 업로드

MIDI 파일을 드래그 앤 드롭 또는 클릭하여 업로드합니다.

비디오 또는 오디오 파일을 동일하게 업로드합니다.
> [!TIP]
> 영상 시작부터 소리가 포함된 파일을 사용하세요. 짧은 노트가 많은 MIDI에서는 초반 무음이 반복될 수 있습니다.

----

> 3️⃣ 재생

재생 버튼을 누르면 MIDI에 맞춰 소리가 재생됩니다.

동시에 영상이 노트 타이밍에 맞춰 제어됩니다.
> [!WARNING]
> MIDI 노트 수가 너무 많거나, 업로드한 영상의 길이가 너무 길 경우 딜레이가 발생할 수 있습니다.  
> 원활한 사용을 위해 적절한 길이의 영상과 단순한 MIDI를 사용하는 것을 권장합니다.

----

> 4️⃣ 영상 효과

좌우 반전 / 상하 반전 기능을 사용할 수 있습니다.

노트가 재생될 때마다 영상에 애니메이션이 적용됩니다.
> [!IMPORTANT]
> 상하 반전 기능과 페이드 아웃 기능은 실험적 기능입니다.  
> 예상과 다르게 동작하거나, 일부 환경에서 정상적으로 작동하지 않을 수 있습니다.

----

> 5️⃣ 결과물 사용

화면 녹화 기능(OBS, Windows 캡처 등)을 이용해 결과를 저장할 수 있습니다.

생성된 영상은 자유롭게 편집 및 업로드가 가능합니다.
> [!NOTE]
> MIDI는 멜로디가 뚜렷한 파일(단선율 위주)을 사용하는 것이 좋습니다.
>
> 이 시스템은 하나의 샘플을 모든 음에 동일한 볼륨으로 적용하여 재생하기 때문에,
> 여러 음(코드)이 동시에 나오면 메인 멜로디가 따로 강조되지 않고 소리가 겹쳐 뭉개질 수 있습니다.
> 
> 오디오는 보컬/드럼이 제거된 상태일수록 결과가 깔끔합니다.

> [!WARNING]
> ### ❌ 사용 시 주의사항 (하면 안 되는 것)
> 
> **🎵 코드(화음) 위주의 MIDI 사용 금지**  
> 여러 음이 동시에 재생되면 소리가 겹쳐 뭉개질 수 있습니다.  
> 단선율(멜로디 중심) MIDI 사용을 권장합니다.
> 
> **⚡ 너무 빠른 MIDI 사용 금지**  
> 노트가 과도하게 많으면 영상 효과가 과하게 발생하여 화면이 불안정해질 수 있습니다.
> 
> **📦 과도하게 큰 파일 업로드 금지**  
> 브라우저 성능 저하 또는 멈춤 현상이 발생할 수 있습니다.
> 
> **🎛 멜로디 자동 강조 기대 금지**  
> 모든 음이 동일한 방식으로 재생되기 때문에 특정 음만 따로 강조되지 않습니다.
> 
> **🎨 색상 기반 효과 기대 금지**  
> 크로마키 영상 사용 시 색상 변화 효과는 의도치 않은 결과를 만들 수 있습니다.
> 
> **📱 모바일에서 고성능 기대 금지**  
> 일부 효과는 모바일 환경에서 기능들이 원활하게 작동하지 않을 수 있습니다.
